# LegalEase AI Frontend

This is the frontend for LegalEase AI - Legal Document Simplifier.

## Getting Started

1. Install dependencies:
   ```
   npm install
   ```

2. Start the development server:
   ```
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Features

- Paste legal documents for analysis
- Upload document files
- View simplified analysis with risky clause highlighting
- Get negotiation suggestions
